Principais formatações

Negrito ou destaque

Nesta frase, temos um termo em negrito usando a tag <b> (não semântica)

Nesta frase temos um termo em destaque usando a tag <strong> (semântica)

Itálico

Nesta frase, temos um termo em Itálico usando a tag <i> (não semântica)

Nesta frase temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos também marcar um texto com a tag MARK

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido para dar ênfase que ele foi adicionado depois

Existe também a tag U que é sublinhado (não semântica)

Texto sobrescrito

Para escrever coisas do tipo x20+3 use a tag SUP

Texto subscrito

Para inserir coisas do tipo H20 use a tag SUB
